HOW IT WORKS:
═══════════════════════════════════════════════════════════════

TRAINING CYCLE FLOW:

  Start Training Cycle
      ↓
  ┌─ Crawl random websites (2 sites)
  │  └─ Max check: if > 5 min, exit
  │
  ├─ Research topics (up to 7 topics)
  │  └─ Max check: if > 5 min, exit
  │
  └─ Minimum Duration Check:
     └─ If < 1 min, continue exploring!
        └─ Pick random new domains
        └─ Keep crawling until ≥ 1 min
        └─ Or hit 5 min limit


ALGORITHM:

1. Start timer
2. Crawl 2 random websites
   ├─ Check max time (5 min)
   └─ Continue if < 5 min
3. Research up to 7 topics
   ├─ Check max time (5 min)
   └─ Continue if < 5 min
4. Check minimum time (1 min)
   ├─ If DONE < 1 min:
   │  └─ Continue exploring new domains
   │     ├─ Keep crawling
   │     └─ Until ≥ 1 min or ≥ 5 min
   └─ If DONE ≥ 1 min:
      └─ Finalize and report


DURATION GUARANTEES:
═══════════════════════════════════════════════════════════════

MINIMUM: 1 minute (60 seconds)
  ✅ If cycle finishes quickly, automatically continues exploring
  ✅ Keeps crawling new domains until 1 min is reached
  ✅ Ensures substantial learning happens
  ✅ Prevents wasted cycles

MAXIMUM: 5 minutes (300 seconds)
  ✅ Hard stop at 5 minutes
  ✅ Doesn't explore new sites after 5 min reached
  ✅ Finishes gracefully at max time
  ✅ Prevents runaway cycles

CODE IMPLEMENTATION:

Added to run_training_cycle():

const min_duration = Duration::from_secs(60);    // 1 minute
const max_duration = Duration::from_secs(300);   // 5 minutes

// During website crawling:
if cycle_timer.elapsed() >= max_duration {
    break;  // Stop if reached max
}

// During topic research:
if cycle_timer.elapsed() >= max_duration {
    break;  // Stop if reached max
}

// After main work:
while cycle_timer.elapsed() < min_duration {
    // Pick random sites and continue exploring
    // until we hit minimum or maximum
}

WHAT HAPPENS IN EXTENDED EXPLORATION:
═══════════════════════════════════════════════════════════════

1. Continues exploring
2. Randomly picks additional sites
3. Crawls them with depth=1
4. Updates reports with new data
5. Continues until:
   ├─ Reaches 1 minute (minimum), OR
   └─ Reaches 5 minutes (maximum)
